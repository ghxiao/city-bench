Experiments for ORE 2014

We conducted the experiments on a HP Proliant Linux server with 144 cores @3.47GHz, 106GB of RAM and a 1TB 15K RPM HD. The systems were evaluated on Java 7 with 8GB memory for the JVM. The timeouts are set to 10min (indicated by ‘-’) for query answering and to 1h for ontology loading.

Spatial Relations Benchmark (BM1)

For the cities of Cork, Riga, Bern, and Vienna, we transform all the amenities (e.g., restaurants), leisure areas (e.g., playgrounds), and shops to concept assertions. Then, we calculate the roles for the spatial relations inside between leisure areas and amenities. Further we generate several next relations between amenities and shops. The different next relations are generated according to the distances of 50m, 100m, 250m, and 500m between two objects.

The OSM instances are available under cities

The instance data can be generated the follwing way using the mapping file from mappings: 
 $ ./generate.py –mappingFile osm_mapping_bm1.txt –mode etl
